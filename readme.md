## Todo-список на Angular Material

### Требования

node >= v6.9

### Установка и запуск

* Клонировать репозитарий.
* ```npm install```.
* Собрать клиент ```gulp```.
* Запустить локальный сервер ```gulp serve```.
* Открыть в браузере [http://localhost:8000/](http://localhost:8000/).
